Henrique AlvesProduct Engineer

Ola! My name is Henrique Alves, I'm Product Engineer. I like product design, UI prototyping, and front-end architecture. Currently prototyping FX/trading tools at Royal Bank of Scotland.

My previous work experience also includes Hailo (2014), The Times (2013-2014), Samsung (2012) and B-Reel (2011).

The front-end stack I'm digging is React, Webpack and PostCSS. Although I've done projects with Ember.js, Backbone, Gulp and Sass. For backend I'm using Node/MongoDB and previously working with Python/Django.

React Inspire

React Inspire is a side-project, a gallery of curated sites built with React. It's an early/alpha version done over the weekend. Built with React, Webpack and CSS Modules. The source code is also available on GitHub.

Hailo

Hailo is a free mobile app for book taxis or private cars. I built the website, the front-end framework for the web apps and the style-guide. We released the first version within 10 weeks. The site is fully static generated using Jekyll, Sass e Node.

Times Collection

Times Collection is a publishing tool for online magazines built initially for The Times iPad app edition. It was an exciting project and opportunity to work closely with the design team helping them with the UX, grid layout and typography. I built whole project in about 4 months, both the Django backend and the front-end using Ember.js
